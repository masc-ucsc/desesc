# SPECSIM.md

This file provides guidance when working with code of the SPECSIM, a Transinet-Safe modeling of DESESC, in this repository.

## Project Overview

DESESC is an architectural simulator for computer architecture research. Key features include out-of-order and multicore modeling, Spectre-Safe modeling, 
       TOML-based configuration, and power/thermal modeling. SPECSIM is a Transinet-Safe modeling of this simulator.

## Build Commands SPECSIM

```bash
# Debug build (development)
 bazel build -c dbg //main:desesc

# Release build (long simulations)
 bazel build -c opt //main:desesc

# Run desesc
 ./bazel-bin/main/desesc -c ./conf/desesc.toml

# Run desesc with printf enable
 ./bazel-bin/main/desesc -c ./conf/desesc.toml >a.txt

# Disable printf using vim 
 vim
 :%s#printf#//printf#gc

# Clean build (if compiler tools change)
 bazel clean --expunge

# Timing Analysis Commands

# Run 2 simulations with same parameters only change do_random = true (Transient/T) and do_random = false (NonTransient/NT) in conf/desesc.toml
  do_random = true
  do_random = false

# Global Time Analysis of the Autogenerated desesc.abc(T) desesc.abc(NT) files
  grep clock desesc.abc
  grep clock desesc.xyz

# Autogenerated files for Transient Simulation(T) and NonTransient Simulations(NT) for Instruction Fetch Timimg Analysis
  time_T.abc time_NT.xyz

# Timing Analysis 
 python3 conf/scripts/diff_time.py time_NT.xyz time_T.abc
